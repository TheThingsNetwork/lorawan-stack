Gateway APIs

The GatewayRegistry service

Method GatewayRegistry.Create

Create a new gateway. This also sets the given organization or user as first collaborator with all possible rights.

Request: CreateGatewayRequest
Response: Gateway

POST /users/{collaborator.user_ids.user_id}/gateways

POST /organizations/{collaborator.organization_ids.organization_id}/gateways

Method GatewayRegistry.Get

Get the gateway with the given identifiers, selecting the fields given by the field mask. The method may return more or less fields, depending on the rights of the caller.

Request: GetGatewayRequest
Response: Gateway

GET /gateways/{gateway_ids.gateway_id}

Method GatewayRegistry.List

List gateways. See request message for details.

Request: ListGatewaysRequest
Response: Gateways

GET /gateways

GET /users/{collaborator.user_ids.user_id}/gateways

GET /organizations/{collaborator.organization_ids.organization_id}/gateways

Method GatewayRegistry.Update

Request: UpdateGatewayRequest
Response: Gateway

PUT /gateways/{gateway.ids.gateway_id}

Method GatewayRegistry.Delete

Request: GatewayIdentifiers
Response: google.protobuf.Empty

DELETE /gateways/{gateway_id}

The GatewayAccess service

Method GatewayAccess.ListRights

Request: GatewayIdentifiers
Response: Rights

GET /gateways/{gateway_id}/rights

Method GatewayAccess.CreateAPIKey

Request: CreateGatewayAPIKeyRequest
Response: APIKey

POST /gateways/{gateway_ids.gateway_id}/api-keys

Method GatewayAccess.ListAPIKeys

Request: ListGatewayAPIKeysRequest
Response: APIKeys

GET /gateways/{gateway_ids.gateway_id}/api-keys

Method GatewayAccess.GetAPIKey

Request: GetGatewayAPIKeyRequest
Response: APIKey

GET /gateways/{gateway_ids.gateway_id}/api-keys/{key_id}

Method GatewayAccess.UpdateAPIKey

Update the rights of an existing gateway API key. To generate an API key, the CreateAPIKey should be used. To delete an API key, update it with zero rights. It is required for the caller to have all assigned or/and removed rights.

Request: UpdateGatewayAPIKeyRequest
Response: APIKey

PUT /gateways/{gateway_ids.gateway_id}/api-keys/{api_key.id}

Method GatewayAccess.GetCollaborator

Get the rights of a collaborator (member) of the gateway. Pseudo-rights in the response (such as the “_ALL” right) are not expanded.

Request: GetGatewayCollaboratorRequest
Response: GetCollaboratorResponse

GET /gateways/{gateway_ids.gateway_id}/collaborator

GET /gateways/{gateway_ids.gateway_id}/collaborator/user/{collaborator.user_ids.user_id}

GET /gateways/{gateway_ids.gateway_id}/collaborator/organization/{collaborator.organization_ids.organization_id}

Method GatewayAccess.SetCollaborator

Set the rights of a collaborator (member) on the gateway. It is required for the caller to have all assigned or/and removed rights. Setting a collaborator without rights, removes them.

Request: SetGatewayCollaboratorRequest
Response: google.protobuf.Empty

PUT /gateways/{gateway_ids.gateway_id}/collaborators

Method GatewayAccess.ListCollaborators

Request: ListGatewayCollaboratorsRequest
Response: Collaborators

GET /gateways/{gateway_ids.gateway_id}/collaborators

Message Gateway

Gateway is the message that defines a gateway on the network.

ids (GatewayIdentifiers)

required

created_at (google.protobuf.Timestamp)
updated_at (google.protobuf.Timestamp)
name (string)

max_len: 50

description (string)

max_len: 2000

attributes (map of string to string)
contact_info (repeated ContactInfo)
version_ids (GatewayVersionIdentifiers)

required

gateway_server_address (string)

pattern: ^(?:(?:[a-zA-Z0-9]|[a-zA-Z0-9][a-zA-Z0-9\-]*[a-zA-Z0-9])\.)*(?:[A-Za-z0-9]|[A-Za-z0-9][A-Za-z0-9\-]*[A-Za-z0-9])(?::[0-9]{1,5})?$|^$

The address of the Gateway Server to connect to. The typical format of the address is “host:port”. If the port is omitted, the normal port inference (with DNS lookup, otherwise defaults) is used. The connection shall be established with transport layer security (TLS). Custom certificate authorities may be configured out-of-band.

auto_update (bool)
update_channel (string)
frequency_plan_id (string)

max_len: 64

antennas (repeated GatewayAntenna)
status_public (bool)

The status of this gateway may be publicly displayed.

location_public (bool)

The location of this gateway may be publicly displayed.

schedule_downlink_late (bool)

Enable server-side buffering of downlink messages. This is recommended for gateways using the Semtech UDP Packet Forwarder v2.x or older, as it does not feature a just-in-time queue. If enabled, the Gateway Server schedules the downlink message late to the gateway so that it does not overwrite previously scheduled downlink messages that have not been transmitted yet.

enforce_duty_cycle (bool)

Enforcing gateway duty cycle is recommended for all gateways to respect spectrum regulations. Disable enforcing the duty cycle only in controlled research and development environments.

downlink_path_constraint (DownlinkPathConstraint)

defined_only

Enums

Enum DownlinkPathConstraint

Name Value Description
DOWNLINK_PATH_CONSTRAINT_NONE 0 Indicates that the gateway can be selected for downlink without constraints by the Network Server.
DOWNLINK_PATH_CONSTRAINT_PREFER_OTHER 1 Indicates that the gateway can be selected for downlink only if no other or better gateway can be selected.
DOWNLINK_PATH_CONSTRAINT_NEVER 2 Indicates that this gateway will never be selected for downlink, even if that results in no available downlink path.
